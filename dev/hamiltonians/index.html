Hamiltonian models

This page lists all concrete Hamiltonians implemented in Damysos. Every model is a two-band Hamiltonian of the form $\hat{H} = \vec{h}(\vec{k})\cdot\vec{\sigma}$ subtyping GeneralTwoBand; the machinery for eigenstates and velocity/dipole matrix elements is described in the two-band formalism.

Damysos.GeneralTwoBand — Type
GeneralTwoBand{T} <: Hamiltonian{T}

Supertype of all 2x2 Hamiltonians with all matrixelements via dispatch.

Idea

The idea is that all Hamiltonians of the form

\[\hat{H} = \vec{h}(\vec{k})\cdot\vec{\sigma}\]

can be diagonalized analytically and hence most desired matrixelements such as velocities or dipoles can be expressed solely through

\[\vec{h}(\vec{k})=[h_x(\vec{k}),h_y(\vec{k}),h_z(\vec{k})]\]

and its derivatives with respect to $k_\mu$. Any particular Hamiltonian deriving form GeneralTwoBand{T} must then only implement $\vec{h}(\vec{k})$ and its derivatives.

The models fall into two groups: periodic models defined on a lattice with a finite Brillouin zone, and nonperiodic continuum models. The distinction matters when choosing a k-space grid — the Simulation constructor enforces the compatible pairing.

Periodic models

Periodic (tight-binding) models are defined on a lattice, so their Brillouin zone is finite and the Hamiltonian is invariant under reciprocal lattice translations. They must be paired with a periodic k-space grid sampling exactly one Brillouin zone, i.e. CartesianMPKGrid1d or HexagonalMPKGrid2d.

Damysos.MonolayerhBN — Type
MonolayerhBN{T<:Real} <: GeneralTwoBand{T}

A tight-binding model for monolayer hexagonal boron nitride (hBN).

The Hamiltonian reads

\[\hat{H} = t[1+2\cos(a k_x/2)\cos(\sqrt{3} k_y a/2)]\sigma_x - 2t \cos(a k_x/2)\sin(\sqrt{3} k_y a/2)\sigma_y + \frac{\Delta}{2}\sigma_z\]

see https://doi.org/10.1103/PhysRevB.83.235312 for details.

Examples

julia> h = MonolayerhBN(0.2,1.0, 1.0)
MonolayerhBN:
  Δ: 0.2
  t: 1.0
  a: 1.0

Damysos.SemiconductorToy1d — Type
SemiconductorToy1d{T<:Real} <: GeneralTwoBand{T}

A toy model for a 1D semiconductor with a direct bandgap at the Brillouin zone center.

The Hamiltonian reads

\[\hat{H} = \Delta\sigma_x - t \sin(a k_x/2)\sigma_z\]

with kx=0 the BZ center. Note that $\Delta$ is half the direct bandgap: the band splitting of a GeneralTwoBand is $2|\vec{h}|$, so the gap at $k_x=0$ is $2\Delta$. Accordingly, the constructor argument halfgap (default 1.65 eV) yields a physical bandgap of 3.3 eV. The default values when given a UnitScaling object (see Examples below) fit bandgap and dipole at $k_x=0$ with $\hat{x}∥\Gamma -M$ to the wurtzite ZnO model used in https://doi.org/10.1103/PhysRevLett.113.073901.

Examples

julia> h = SemiconductorToy1d(UnitScaling(u"1.0fs",u"1.0Å"))
SemiconductorToy1d:
  Δ: 2.507
  t: 6.533
  a: 2.82

Nonperiodic models

Nonperiodic models describe a continuum approximation around a band extremum; k-space is unbounded and the simulated region is chosen large enough for the physics at hand. They are used with the aperiodic Cartesian k-grids such as CartesianKGrid1d or CartesianKGrid2d.

Damysos.GappedDirac — Type
GappedDirac{T<:Real} <: GeneralTwoBand{T}

Massive Dirac Hamiltonian (two-band model).

The Hamiltonian reads

\[\hat{H} = k_x\sigma_x + k_y\sigma_y + m\sigma_z\]

such that $\vec{h}=[k_x,k_y,m]$.

Examples

julia> h = GappedDirac(1.0)
GappedDirac:
  m: 1.0
  vF: 1.0

Damysos.QuadraticToy — Type
QuadraticToy{T<:Real} <: GeneralTwoBand{T}

A toy model of quadratic bands with a gap.

The Hamiltonian reads

\[\hat{H} = \frac{\zeta}{2}(k_x^2\sigma_x + k_y^2\sigma_y) + \frac{\Delta}{2}\sigma_z\]

such that $\vec{h}=[\zeta/2 k_x^2,\zeta/2 k_y^2, \Delta/2]$. The dimensionful form (SI) would be

\[\hat{H}_{SI} = \frac{\hbar^2}{2m^*}(k_x^2\sigma_x+k_y^2\sigma_y)+\frac{E_{gap}}{2}\sigma_z\]

Examples

julia> h = QuadraticToy(0.2,1.0)
QuadraticToy:
  Δ: 0.2
  ζ: 1.0

Damysos.BilayerToy — Type
BilayerToy{T<:Real} <: GeneralTwoBand{T}

A toy model of touching quadratic bands with a gap turning them quartic near the band edge.

The Hamiltonian reads

\[\hat{H} = \frac{\zeta}{2}[(k_y^2-k_x^2)\sigma_x - 2 k_x k_y \sigma_y] + \frac{\Delta}{2}\sigma_z\]

such that $\vec{h}=[\zeta/2 (k_y^2-k_x^2), -\zeta k_x k_y, \Delta/2]$. The dimensionful form (SI) would be

\[\hat{H}_{SI} = \frac{\hbar^2}{2m^*}[(k_y^2-k_x^2)\sigma_x - 2 k_x k_y \sigma_y]+\frac{E_{gap}}{2}\sigma_z\]

Examples

julia> h = BilayerToy(0.2,1.0)
BilayerToy:
  Δ: 0.2
  ζ: 1.0
